
在某些前端项目开发过程中，我们需要不断在微信中调试和测试我们的页面，这给本地调试造成了很大的不便。
Weixin-WebView-Mock就是解决这个问题的迷你辅助工具，正所谓：

```
专注于小而美，令人心驰神往
```

__功能特性__

1. Mock了WeixinJSBridge的on, invoke, call等核心接口
2. 独立app方式运行，可在真机上调试页面
3. 支持ios和android
4. 微信内置webview的界面和交互模拟
5. 更多功能和特性加入中
